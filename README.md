# Academic Pro - Gestion du Pôle Digital

Academic Pro est une plateforme d'excellence pédagogique conçue pour la gestion et le suivi des activités au sein du Pôle Digital.

## Caractéristiques
- **Directeur Pédagogique (DP)**: Validation des présences, statistiques globales, gestion des groupes et formateurs.
- **Formateur**: Saisie de l'avancement, notation des stagiaires, consultation de l'emploi du temps.
- **Stagiaire**: Consultation des notes, suivi des présences, accès à l'emploi du temps hebdomadaire.

## Technologies
- Flutter (Web, Android, iOS)
- SQLite (Persistance locale)
- Provider (Gestion d'état)

## Installation
1. `flutter pub get`
2. `flutter run`
